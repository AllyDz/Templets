Link Description
<' a '> element to define a hyperlink to a section (bookmark) of a page via a #id or to another page

Attribute/Element Description
href attribute to define the link address
target attribute to define where to open the linked document
<' img '> element (inside ) to use an image as a link
id attribute (id="value") to define bookmarks in a page

a:hover When the mouse is over the link text is will have a different color displayed
a:active When a link has been activated it will have a different color displayed

JavaScript Introduction

Document.getElementById is one of the methods to make changes to your HTML content. Ex:document.getElementById("your id here").innerHTML = "Hello JavaScript"; changes the current text assigned to that id to "Hello JavaScript"

Examples Here

Where To Place Your JavaScript

You can have an external JS file, or place your JS in the <' body '>' and/or the <' head '> tags.
In your HTML you need to place your JS file link or in-line scripts within <' script '> tags

External JavaScripts files have some advantages:
-Separates HTML and code
-Makes HTML and JavaScript easier to read and maintain
-Cached JavaScript files can speed up page loads
